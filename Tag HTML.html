# Tag Mean Code
1 h1,h2,h3,h4,h5,h6 ขนาดแตกต่างกันและเว้นบรรทัด < h1 > ... < /h1 >
2 br ใช้เว้นบรรทัดใหม่ < br >
3 p ใช้เว้นบรรทัดใหม่ < p > ... < /p >
4 hr เป็นการขีดเส้นระหว่างบรรทัด < hr >
5 b ทำให้ข้อความเป็นตัวหนา < b > ... < /b >
6 strong ทำให้ข้อความเป็นตัวหนา < strong > ... < /strong >
7 pre แสดงข้อความตามที่เราต้องการ < pre > ... < /pre >
8 i ทำให้ข้อความเป็นตัวเอียง < i > ... < /i >
9 em ทำให้ข้อความเป็นตัวเอียง < em > ... < /em >
10 ul เป็นการทำให้ข้อความเป็นข้อๆ < ul > ... < /ul >
11 li เป็นการทำให้ข้อความเป็นข้อๆ < li > ... < /li >
12 ol ทำให้หัวข้อเป็นเลข < ol > ... < /ol >
13 img src="xxx.png" เป็นการใส่รูปที่เราต้องการ < img src="ชื่อไฟล์ในามสกุล" >
14 !--...-- ทำให้ไม่แสดงใน Web !--...--
15 a เป็นการเชื่อมไปยังลิงค์อื่น < a href="ลิงค์ที่ต้องการ" > ... < /a >
16 table สร้างตาราง < table > ... < /table >
17 div เป็นการใส่ Block-level < div > ... < /div >
18 span,sp ทำให้เปลี่ยนสีตัวอักษรได้ < span >... < /span >,< sp > ... < /sp >
19 style ทำให้เปลี่ยนสีตัวอักษรได้ < p style> ... < /p >
20 label ตัวรับ input < label > ... < /label >
21 input การรับข้อมูลเข้า < input type="text" >
22 textarea เป็นtextที่สามารถขยายพื้นที่คำตอบได้ < textarea name="..." cols="..." rows="..." > < /textarea >
23 input type="password" ข้อมูลที่เราคีย์จะมองไม่เห็น < input type="password" >
24 input type="email" เป็นการรับข้อมูลแบบ email < input type="email" >
25 input type="date" เป็นการรับข้อมูลวันเดือนปีเกิด < input type="date" >
26 input type="number" เป็นการรับข้อมูลแบบตัวเลขเท่านั้น < input type="number" >
27 input type="radio" เป็นการรับข้อมูลเข้าแบบตัวเลือก < input type="radio" >
28 input type="tel" เป็นการรับข้อมูลเข้าแบบตัวเลขเท่านั้น < input type="tel" >
29 select เป็นการรับข้อมูลอย่างใดอย่างหนึ่ง < select name=".." > ... < /select >
30 input type="checkbox" เป็นการรับข้อมูลเข้าได้หลายอย่าง < input type="checkbox" >
31 input type="file" สามารถรับข้อมูลเป็นไฟล์ได้ < input type="file" >
32 input type="submit" ส่งข้อมูลที่กรอกข้างต้น < input type="submit" >
33 input type="reset" ลบข้อมูลทั้งหมดที่กรอกมา < input type="reset" >
